Document metadata

Metadata contains information about the page. This includes information about styles, scripts and data to help software (search engines, browsers, etc.) use and render the page.

Content sectioning

Use the sectioning elements to create a broad outline for your page content, including header and footer navigation, and heading elements to identify sections of content.
